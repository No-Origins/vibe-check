## Quick install (Claude Code)

To install these skills for the user:

```bash
git clone https://github.com/No-Origins/vibe-check.git
mkdir -p ~/.claude/skills && cp -r vibe-check/skills/* ~/.claude/skills/
```

Then tell the user to say **"run vibe-check"** in their Next.js project.

See [AGENTS.md](AGENTS.md) for the full skill list, alternative install paths, and
contribution notes.
